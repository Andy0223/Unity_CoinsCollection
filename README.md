## 簡要說明遊戲操作
- 按 D 鍵 或右鍵可右移玩家
- 按 A 鍵 或左鍵可左移玩家
- 按空白鍵即可跳躍（連續點可多段跳躍）

## 說明主要遊戲機制
- 遊戲初始會給 10 秒的遊戲時間，接著遊戲時間便開始倒數，過程中透過移動主角，藉由與金幣的碰撞偵測，取得額外秒數（每個金幣可取得 2 秒），須在遊戲時間內收集完所有金幣，否則失敗。
- 若在遊戲過程中碰到粉紅色的尖刺（Spikes），一樣遊戲失敗
